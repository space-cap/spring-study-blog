


## STS4(Spring Tool Suite 4)에서 자주 사용하는 단축키 정리

**저장 및 편집**
- Ctrl + S : 저장
- Ctrl + Z / Ctrl + Y : 실행 취소 / 다시 실행
- Ctrl + X / Ctrl + C / Ctrl + V : 잘라내기 / 복사 / 붙여넣기
- Ctrl + D : 한 줄 삭제
- Alt + ↑ / Alt + ↓ : 한 줄 위/아래로 이동
- Ctrl + Alt + ↑ / Ctrl + Alt + ↓ : 한 줄 복제

**코드 자동화 및 정리**
- Ctrl + Shift + F : 코드 자동 정렬(포맷팅)
- Ctrl + Shift + O : import 추가 및 정리
- Ctrl + Space : 자동 완성(컨텐트 어시스트)
- Ctrl + 1 : 퀵 픽스(에러/경고 자동 해결 제안)
- Alt + Shift + R : 동일 변수명 한 번에 변경(리네임)
- Alt + Shift + S : 소스 생성(게터/세터/생성자 등)

**주석 처리**
- Ctrl + / 또는 Ctrl + Shift + C : 한 줄 주석/해제
- Ctrl + Shift + / : 블록 주석 설정
- Ctrl + Shift + \ : 블록 주석 해제

**검색 및 탐색**
- Ctrl + F : 현재 파일 내 검색
- Ctrl + H : 전체 프로젝트 내 검색
- Ctrl + Shift + R : 파일 이름으로 검색 및 열기
- Ctrl + Shift + L : 모든 단축키 목록 보기

**코드 이동 및 탐색**
- F3 : 선언(정의)로 이동
- Ctrl + 클릭 : 선언(정의)로 이동
- Alt + ← / Alt + → : 이전/다음 위치로 이동
- Ctrl + Q : 마지막 편집 위치로 이동
- Ctrl + L : 지정한 줄 번호로 이동
- Ctrl + E : 열려있는 에디터 목록 보기
- Ctrl + PgUp / Ctrl + PgDn : 열려있는 파일 간 이동

**실행 및 디버깅**
- Ctrl + F11 : 프로젝트 실행
- F11 : 디버그 실행
- Ctrl + Shift + B : 브레이크포인트 설정/해제

**기타 유용한 단축키**
- Ctrl + Shift + L : 단축키 전체 목록 확인
- Ctrl + N : 새 파일/프로젝트 생성
- Ctrl + M : 에디터 최대/최소화
- Alt + Shift + A : 다중 열 편집 모드

이 외에도 다양한 단축키가 있으니, **Ctrl + Shift + L**을 눌러 전체 단축키 목록을 확인하며 본인에게 유용한 단축키를 익혀두면 개발 효율이 크게 향상됩니다[1][2][3][5][6][7].



STS4(Spring Tool Suite 4)에서 코드 정리(자동 정렬) 단축키는 **Ctrl + Shift + F**입니다. 이 단축키를 누르면 자바, HTML, JSP 등 다양한 파일에서 코드가 자동으로 정렬됩니다[1][2][3][4][5][8].

- **Ctrl + Shift + F** : 코드 자동 정렬(포맷팅)

이 기능은 이클립스(Eclipse)에서도 동일하게 동작하며, STS4 역시 이클립스 기반이기 때문에 같은 단축키를 사용합니다.  
HTML, JSP 파일의 경우 정렬 방식이 마음에 들지 않는다면, 환경설정(Preferences)에서 세부 포맷 옵션을 조정할 수 있습니다[2][6].

STS4(Spring Tool Suite 4)에서 메서드 이름을 클릭하거나 커서를 올린 뒤 해당 메서드의 정의(선언)로 이동하는 기본 단축키는 **F3**입니다.  
또한, 마우스로 메서드에 커서를 올리고 **Ctrl(컨트롤) 키를 누른 상태에서 클릭**해도 동일하게 정의로 이동할 수 있습니다[1][2][4][7].

- **F3** : 커서가 위치한 메서드/변수/클래스의 선언(정의)로 이동
- **Ctrl + 클릭** : 마우스로 해당 요소의 선언(정의)로 이동

이 두 가지 방법 모두 이클립스 기반의 STS에서 널리 사용되는 대표적인 이동 단축키입니다.

STS4(Spring Tool Suite 4)에서 이전에 있던 편집 위치로 돌아가는 단축키는 **Alt + ← (왼쪽 화살표)**입니다.  
반대로 다시 앞으로 이동할 때는 **Alt + → (오른쪽 화살표)**를 사용합니다.

- 이전 위치로 이동 : **Alt + ←**  
- 다음 위치로 이동 : **Alt + →**

이 단축키는 코드 탐색 중에 이전에 보던 위치로 빠르게 돌아갈 때 유용합니다[1][5].
